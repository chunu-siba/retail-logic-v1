# Python-Driven Retail Simulation & Budget Logic Engine Repository Name: retail-logic-v1
A command-line interface (CLI) application designed to simulate the end-to-end retail experience, from inventory browsing to budget-constrained checkout. This project focuses on complex state management and user-input validation in Python.Key Features: * Dynamic Cart Management: Implemented CRUD (Create, Read, Update, Delete) functionality for a virtual shopping cart.Budget Guardrails: Real-time budget tracking with automated "Over-limit" intervention logic.Defensive Programming: Extensive use of try-except blocks and input sanitization to ensure system stability against "dirty" user data.Data Structures: Leveraged dictionaries and tuples for $O(1)$ and $O(n)$ lookup efficiencies in inventory management.
